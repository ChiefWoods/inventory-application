# App

## Setup

1. Install all dependencies

```bash
bun install
```

2. Create env file

```bash
cp .env.example .env.development
```

Set `VITE_API_URL` to your running API URL (default local API is `http://localhost:3000`).

3. Start development server

```bash
bun run dev
```

4. Build project

```bash
bun run build
```

5. Preview build

```bash
bun run preview
```

## Testing

1. Test project

```bash
bun run test
```
